# BuildSimulationInput マクロ 導入ガイド

## 概要

**BuildSimulationInput** VBAマクロは、仕様書「16. シミュレーション入」の「データセット仕様（追加）」を実装したものです。

設定シートの「開始年月」「計画月数」に基づいて、生産計画データを自動的に横持ち形式に変換し、シミュレーション入シートを再構築します。

---

## 対応仕様

### 仕様書：16. シミュレーション入 - データセット仕様（追加）

```
1. 設定シートから開始年月と計画月数を取得する
2. 開始年月を起点に、計画月数分の連続年月列をC列以降に作成する
3. 生産計画シートから、対象年月が上記期間に含まれるデータのみ抽出する
4. 品目コードごとに横持ち化し、シミュレーション入へセットする
5. 対象期間外の年月はシミュレーション入にセットしない
```

## インストール方法

### 方法1: VBA編集画面へ手動貼り付け

**ファイル**: `BuildSimulationInput.vba`

1. Excelで `BOM copy 1.xlsm` を開く
2. **Alt+F11** キーを押してVBA編集画面を開く
3. メニュー「挿入」→「標準モジュール」を選択
4. `BuildSimulationInput.vba` の内容全体をコピー
5. VBA編集画面の白い編集領域に貼り付け
6. **Ctrl+S** で保存
7. VBA編集画面を閉じる（**Alt+Q**）

### 方法2: Excel開発者タブから（Windows）

1. `BOM copy 1.xlsm` を開く
2. 「ファイル」→「オプション」→「リボンのユーザー設定」
3. 「開発者」にチェックを入れて「OK」
4. リボンに「開発者」タブが表示される
5. 「開発者」→「Visual Basic」をクリック
6. 上記方法1の手順3以降を実行

|------|------|------|-----|

## エラーが発生した場合

### エラー: シートが見つかりません

**原因**: シート名が仕様と異なっている

**対処**: 以下のシート名が存在することを確認
- `設定`
- `生産計画`
- `シミュレーション入`
- `品目マスタ`

シート名を変更するか、VBAコードの以下の行を修正してください：
```vba
Set wsConfig = ThisWorkbook.Sheets("設定")
Set wsPlan = ThisWorkbook.Sheets("生産計画")
Set wsIn = ThisWorkbook.Sheets("シミュレーション入")
Set wsMaster = ThisWorkbook.Sheets("品目マスタ")
```

### エラー: 型が一致しません

**原因**: 設定シートのB1またはB2が数値ではない

**対処**: 
- B1に6桁の数字（例: 202605）を入力
- B2に月数の数字（例: 3）を入力
- セルを「数値」書式に設定

### エラー: 指定されたセルはありません

**原因**: 生産計画または品目マスタが空

**対処**:
- 生産計画シートに品目コード、対象年月、計画数量が入力されていることを確認
- 品目マスタに品目コードと品目名が入力されていることを確認

## 処理内容

### 実行順序

1. **入力値の読み込み**
   - 設定シート: B1（開始年月）、B2（計画月数）

2. **月列の生成**
   - 開始年月から計画月数分の連続月を生成
   - 例: 202605 → [202605, 202606, 202607]

3. **生産計画データの抽出**
   - 生産計画シートから対象期間のデータをフィルタ
   - 品目コード毎に月次数量を集計

4. **品目マスタの参照**
   - 品目コード → 品目名をマッピング

5. **シミュレーション入の再構築**
   - 既存データを削除
   - ヘッダー行を作成（品目コード、品目名、月列）
   - 品目ごとに月次数量を横持ち化
   - 品目コード昇順でソート
